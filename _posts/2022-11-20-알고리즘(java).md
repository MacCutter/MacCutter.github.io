## 프로그래머스 알고리즘 같은 숫자는 싫어 문제

<img src="문제.png">

```java
import java.util.*;

public class Solution {
    public int[] solution(int []arr) {

        ArrayList<Integer> answer = new ArrayList<>();

        answer.add(arr[0]);

        for (int i = 1; i < arr.length; i++) {
            if (arr[i - 1] != arr[i]) {
                answer.add(arr[i]);
            }
        }
        return answer.stream().mapToInt(i -> i).toArray();
    }
}
```
